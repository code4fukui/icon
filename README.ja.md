# icon

福井県が公開しているオープンデータのアイコンとイラストのコレクションです。

## 特徴

- オープンデータのアイコンとイラストの包括的なセットです。
- 福井県のオープンデータの取り組みから直接提供されています。
- 個人利用および商用利用のどちらでも無料で利用できます。

## 使用方法

アイコンを使用するには、このリポジトリから必要なファイルを直接ダウンロードしてください。

## データソース

このプロジェクトは、福井県が公開しているアイコンとイラストのオープンデータを利用しています。
