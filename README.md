# icon

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A collection of open data icons and illustrations from FUKUI Prefecture, Japan.

## Features

- A comprehensive set of open data icons and illustrations.
- Sourced directly from FUKUI Prefecture's open data initiatives.
- Free for both personal and commercial use.

## Usage

To use the icons, download the desired files directly from this repository.

## Data Source

This project utilizes the icon and illustration open data published by FUKUI Prefecture.
